Search
Two behaviors of the search bar are available.
  1. Collapsible Search Button: When the user clicks on the search button, the box expands and the cursor is focused into the search field. Once the user loses focus on the search box, the box collapses into a button.
  2. Expanded Search Bar: When a user focuses within the search bar, an “X” appears on the right side of the search bar.

Toast Notifications

Toast notifications appear below the main navigation bar

Confirmation toast auto-dismisses after 5 seconds or after the user dismisses the message by clicking on DISMISS.
Error toast does not auto-dismiss and must be dismissed by the user or until they navigate away from the page.
When additional error information is returned by the API, the error toast should display a user friendly message with further details so the user can resolve the error.

MDC Toolbar acts as a container for multiple rows containing items such as application title, navigation menu
MDC Toolbar acts as a container for multiple rows containing items such as application title, navigation menu, and tabs, among other things. Toolbars scroll with content by default, but support fixed behavior as well.
Flexible behavior can be added to mdc-toolbar, where the height of the toolbar changes as the user scrolls.
