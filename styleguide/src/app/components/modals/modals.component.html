Modals

Modal dialogs should be used in the event the user needs to confirm a selection or perform additional actions. Modals should not be used as a notification.

Dismissing a modal dialog should provide a multi-method approach:

  1. Primary call-to-action
  2. Cancel secondary call-to-action 2.1 In-line Cancel confirmation
